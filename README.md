# YouTubeToMp3

YouTubeToMp3 is a simple tool that converts YouTube videos into audio (MP3) files. It is designed to be easy to use and lightweight, making it ideal for quickly extracting audio from videos for personal use.

## Features

* Convert YouTube videos to MP3 audio
* Simple and fast conversion process
* Lightweight and easy to run
* Works with individual YouTube video URLs

## Requirements

* Internet connection
* YouTube video URL

## Usage

1. Run the application:

2. Paste a YouTube video URL when prompted.

3. The audio will be downloaded and saved as an MP3 file in the output directory.

## Output

Converted MP3 files will be saved locally, typically in an `output` or `downloads` folder (depending on configuration).
